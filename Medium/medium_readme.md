🔹 題目：陣列排序（Bubble Sort）

題目說明
寫一個程式，輸入一組整數陣列，使用 冒泡排序 (Bubble Sort) 將陣列由小到大排序並輸出。
冒泡排序核心概念：

每輪比較相鄰元素，若前者大於後者則交換位置。

每一輪將最大的元素「冒泡」到陣列尾端。

重複直到陣列完全排序。

範例輸入 / 輸出
輸入：[5, 3, 8, 4, 2]
輸出：Sorted Array: 2 3 4 5 8
檔名：bubble_sort.c
